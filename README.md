# 🌾 Digital Agriculture Management System

> **A fully object-oriented desktop application that digitizes farm management — delivering real-time yield predictions, soil-aware recommendations, and investment analytics through a clean Tkinter GUI.**

---

## ⚡ What This Does

Traditional farming runs on guesswork and paper. This system replaces both.

Built in **Python 3** using pure **OOP architecture**, it gives farmers a structured digital platform to track crops, predict harvest yields, get tailored recommendations based on soil and weather conditions, and export actionable insights — all from a tabbed desktop GUI.

---

## 🧠 Core Technical Highlights

**Full OOP Architecture**
Every component is a class with a single responsibility:
- `Crop` — models crop data with encapsulated attributes
- `FarmManager` — manages collections of crop objects
- `Insights` — static methods for yield prediction and recommendations
- `AgricultureSystemGUI` — handles all Tkinter interface logic

**Predictive Yield Engine**
Scores crop yield out of 100 based on soil quality, weather forecast, health status, and soil type — giving farmers a quantifiable output to plan around.

**Smart Recommendation System**
Generates crop-specific guidance covering water needs, fertilizer levels, optimal planting windows, pest threats, preventive measures, and best selling periods.

**Investment Cost Breakdown**
Calculates and itemizes total farming cost across seeds, fertilizers, labor, irrigation, and pesticides — based on land area input.

**Export Functionality**
Recommendations can be exported to a `.txt` file directly from the app — no manual note-taking needed.

## 🖥️ GUI Tabs

| Tab | What it does |
|-----|-------------|
| **Add Crop** | Input crop name, planting date, soil quality, weather, health, soil type, land area |
| **View Crops** | Live table of all crops with growth stage tracking |
| **Insights** | Yield prediction score, cost breakdown, recommendations, export |
| **Help** | Usage guide for the application |
